# /drift-check - 偏离检查

检查当前工作是否偏离了北极星目标。

## 执行步骤

1. 读取 `.ai/NORTH_STAR.md` 文件
2. 总结最近 5 轮对话做了什么
3. 对比分析：
   - 哪些工作推进了北极星目标？
   - 哪些工作偏离了北极星目标？
   - 是否触犯了"绝对不做"的红线？
4. 给出偏离度评估（0-100%）
5. 如果偏离严重，建议下一步行动

## 输出格式

```
## 偏离检查报告

### 北极星
[显示北极星内容]

### 最近工作
- [总结最近做了什么]

### 对齐度评估
- 推进目标：[列出]
- 偏离目标：[列出]
- 触犯红线：[是/否]

### 偏离度：X%

### 建议
[如果偏离严重，给出建议]
```
